# Gait Validation Levels

MVP validation levels:

- L0: single-leg IK passes.
- L1: four-leg phase pattern is complete.
- L2: stand stability meets target duration.
- L3: flat walk has no fall.
- L4: posture stays within roll/pitch limits.
- L5: foot slip stays below target.
- L6: joint torque margin stays above target.
- L7: speed and cost-of-transport are acceptable.

Later levels can add slope, step obstacle, push disturbance, friction variation,
and low-battery derating.
