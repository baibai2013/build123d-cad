# CNC 加工设计规则

> 面向 CNC 铣削/车削的设计约束，确保 build123d 生成的零件可加工。

---

## 1. 刀具可达性

| 规则 | 推荐值 | 说明 |
|------|--------|------|
| 最小内圆角 | = 刀具半径（通常 R1~R3） | 内角必须有圆角，刀具无法切出直角 |
| 最小槽宽 | = 刀具直径 + 0.5mm | 确保刀具能进入 |
| 最小孔径 | ≥ 1mm | 微型钻头容易断 |
| 最大深径比（盲孔） | ≤ 4:1 | 深孔需特殊工艺 |
| 最大深径比（通孔） | ≤ 10:1 | 深孔钻加工 |

### build123d 中设计内圆角

```python
# CNC 友好：所有内角加圆角
with BuildPart() as pocket_part:
    Box(60, 40, 20)
    with BuildSketch(pocket_part.faces().sort_by(Axis.Z)[-1]):
        Rectangle(40, 25)
    extrude(amount=-12, mode=Mode.SUBTRACT)
    
    # ⚠️ CNC 必须：内角圆角 = 刀具半径
    tool_radius = 1.5  # R1.5 刀具
    internal_edges = pocket_part.edges().filter_by(Axis.Z)
    fillet(internal_edges, radius=tool_radius)

export_step(pocket_part.part, "cnc_pocket.step")
```

## 2. 壁厚与深宽比

| 约束 | 铝合金 | 钢 | 塑料 |
|------|--------|-----|------|
| 最小壁厚 | 0.5mm | 0.8mm | 1.0mm |
| 最大深宽比（槽） | 4:1 | 3:1 | 3:1 |
| 最大薄壁高度 | 壁厚 × 20 | 壁厚 × 15 | 壁厚 × 10 |

---

## 3. 拔模角

| 场景 | 推荐拔模角 | 说明 |
|------|----------|------|
| CNC 铣削 | 不需要 | 刀具可切直壁 |
| 压铸件 | 1-3° | 脱模必须 |
| 注塑件 | 0.5-2° | 脱模必须 |
| 砂铸件 | 2-5° | 起模必须 |

```python
# 带拔模角的拉伸（用于铸造/注塑设计）
extrude(amount=20, taper=2)  # 2° 拔模角
```

## 4. 螺纹设计

| 规格 | 钻孔直径 | 最小螺纹深度 | 最小壁厚 |
|------|---------|------------|---------|
| M3 | Ø2.5mm | 6mm (2D) | 3mm |
| M4 | Ø3.3mm | 8mm (2D) | 4mm |
| M5 | Ø4.2mm | 10mm (2D) | 5mm |
| M6 | Ø5.0mm | 12mm (2D) | 6mm |
| M8 | Ø6.8mm | 16mm (2D) | 8mm |

> 注：build123d 不直接生成螺纹几何。导出 STEP 后在 CAM 软件中指定攻丝操作。
> 设计时只需保证底孔直径和深度正确。

## 5. 表面粗糙度

| 加工方式 | 典型 Ra | 适用场景 |
|---------|---------|---------|
| 粗铣 | Ra 3.2~6.3 μm | 非配合面 |
| 精铣 | Ra 0.8~1.6 μm | 一般配合面 |
| 磨削 | Ra 0.2~0.4 μm | 精密配合面 |
| 抛光 | Ra < 0.1 μm | 密封面/光学面 |

## 6. STEP 导出（CAM 适配）

```python
# CNC 加工用 STEP — 最高精度
export_step(part.part, "cnc_part.step",
            precision_mode=PrecisionMode.GREATEST)
```

---

## 7. CNC 设计检查清单

| 检查项 | 通过标准 |
|--------|---------|
| 内圆角 | 所有内角 ≥ 目标刀具半径 |
| 深径比 | 盲孔 ≤ 4:1，通孔 ≤ 10:1 |
| 壁厚 | ≥ 材料最小壁厚 |
| 刀具可达 | 所有加工面可被标准刀具到达 |
| 装夹面 | 至少一个平面可作为装夹基准 |
| 导出格式 | STEP（不是 STL） |
