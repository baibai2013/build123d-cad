# Catalog Rules

The MVP catalog is offline and curated for robot-dog prototypes. Scoring favors:

- matching category and interface
- enough voltage/current margin
- matching package preference
- `jlcpcb_basic` assembly tier
- active lifecycle and stock status not marked unavailable

Blockers:

- No candidate for a required category.
- Best candidate lacks required voltage/current capability.
- Best candidate is obsolete or unavailable.

Warnings:

- Candidate is `jlcpcb_extended`.
- Candidate stock is unknown.
- Selection is from offline metadata only.
