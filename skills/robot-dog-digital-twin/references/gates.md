# Gates

The MVP gate sequence is:

| Gate | Name | Purpose |
|---|---|---|
| G0 | requirements_ready | Requirements and verification matrix exist. |
| G1 | virtual_architecture_ready | Architecture and primary artifacts are declared. |
| G2 | digital_prototype_ready | CAD/PCB/robot model/simulation artifacts are present. |
| G3 | multi_domain_validation_passed | No blocker remains in core validation domains. |
| G4 | iteration_score_passed | Score passes the prototype threshold. |
| G5 | physical_prototype_allowed | G0-G4 pass and manufacturing artifacts exist. |

Score is advisory. Blockers fail gates.
