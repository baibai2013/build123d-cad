# Failure Taxonomy

Severity levels:

- `blocker`: prevents the next gate.
- `major`: serious risk but not always gate-blocking.
- `minor`: should be tracked.

Common blocker IDs:

- `missing_requirements`
- `missing_verification_matrix`
- `missing_artifact`
- `mechanical_interference`
- `pcb_fit_clearance_low`
- `circuit_check_failed`
- `joint_torque_margin_low`
- `stand_stability_failed`
- `flat_walk_no_fall_failed`
- `score_below_threshold`
